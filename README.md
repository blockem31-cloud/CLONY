# 🌀 Clony - Discord Server Kloner

Ein reines **PowerShell** Discord Server Klon-Tool mit verschlüsselter Token-Speicherung, UTF-8 Emoji-Unterstützung und vollständigen Server-Migrationsfunktionen.

---

---

## ✨ Funktionen

- 🔐 **Verschlüsselter Token**: Token wird in einer benutzerspezifischen, verschlüsselten Umgebungsvariable gespeichert.
- 🎨 **Server Icon**: Automatischer Download & Upload.
- 👥 **Rollen**: Alle Rollen mit Berechtigungen, Farben, Positionen.
- 📁 **Kategorien**: Channel-Kategorien werden übernommen.
- 💬 **Text Channels**: Topic, NSFW, Slowmode inklusive.
- 🔊 **Voice Channels**: Bitrate und Benutzer-Limit erhalten.
- 🔐 **Berechtigungen**: Channel-Overwrites korrekt zugeordnet.
- 😀 **Emojis**: Custom Server-Emojis werden geklont.
- 🆕 **Auto-Erstellen**: Ziel-Server automatisch generieren.
- 📊 **Fortschrittsbalken**: Echtzeit-Klon-Fortschritt.
- 🔄 **Auto-Retry**: Behandelt Rate-Limits automatisch.

---

## 📋 Voraussetzungen

- Windows 10/11
- PowerShell 5.1+
- Internetverbindung
- Discord Token (User oder Bot Token)
- Berechtigungen auf Ziel-Server:
  - Server verwalten
  - Rollen verwalten
  - Channels verwalten
  - Emojis verwalten

## 🚀 Installation

### 1. Datei herunterladen

Speichere die Datei **Clony.ps1** lokal auf deinem PC.  

Füge die datei zum path hinzu und gebe anschließend in der powershell Clony ein






#Bedienung
Token eingeben (Option 1)

Gib deinen Discord Token ein.

Clony speichert diesen verschlüsselt in einer Umgebungsvariable.

Beim nächsten Start wird er automatisch geladen.

Server laden (Option 2)

Ruft alle Server ab, auf die dein Account Zugriff hat.

Quelle wählen (Option 3)

Wähle den Server, der geklont werden soll (Vorlage).

Ziel wählen (Option 4)

Wähle den Server, der überschrieben wird (Ziel).

⚠️ WARNUNG: Alle bestehenden Daten werden gelöscht!

Klonen starten (Option 5)

Tippe CLONY zur Bestätigung.

Beobachte den Fortschritt.

Server automatisch erstellen (Option 6)

Wenn kein Ziel-Server existiert, erstellt Clony automatisch einen neuen Server.
